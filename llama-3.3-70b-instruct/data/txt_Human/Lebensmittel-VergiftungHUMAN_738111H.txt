**Lebensmittel-Erkrankungen**

Dieser Text informiert über Erkrankungen, die durch Lebensmittel verursacht werden.

**Was ist der Unterschied zwischen Vergiftung und Infektion?**

Eine **Vergiftung** (das bedeutet, dass Giftstoffe im Körper sind) und eine **Infektion** (das bedeutet, dass Bakterien oder Viren im Körper sind) sind zwei verschiedene Arten von Erkrankungen.

**Lebensmittel-Vergiftung**

Eine Lebensmittel-Vergiftung ist eine Erkrankung, die durch Giftstoffe im Essen verursacht wird. Diese Giftstoffe können von **Bakterien** (kleine Lebewesen, die Krankheiten verursachen können) oder **Schimmelpilzen** (kleine Lebewesen, die auf feuchten Oberflächen wachsen) stammen.

**Lebensmittel-Infektion**

Eine Lebensmittel-Infektion ist eine Erkrankung, die durch Bakterien oder **Viren** (kleine Lebewesen, die Krankheiten verursachen können) im Essen verursacht wird.

**Symptome**

Eine Lebensmittel-Erkrankung kann durch folgende Symptome erkennbar sein:

* **Übelkeit** (das Gefühl, dass man sich übergeben muss)
* **Bauchschmerzen** (Schmerzen im Bauch)
* **Erbrechen** (das Herauswerfen von Speise)
* **Durchfall** (weicher, wässriger Stuhl)

**Warum ist eine Lebensmittel-Erkrankung gefährlich?**

Eine Lebensmittel-Erkrankung kann sehr gefährlich sein, insbesondere für Menschen mit einem **schwachen Immunsystem** (das bedeutet, dass der Körper nicht gut gegen Krankheiten kämpfen kann).

**Wie kann man eine Lebensmittel-Erkrankung vermeiden?**

Sie können das Risiko einer Lebensmittel-Erkrankung verkleinern, indem Sie:

* Lebensmittel ausreichend lange erhitzen
* Ihre Hände gründlich waschen
* Lebensmittel getrennt voneinander verarbeiten
* Die Arbeitsfläche gründlich reinigen

**Wo kann man noch mehr Informationen erhalten?**

Wenn Sie denken, dass Sie eine Lebensmittel-Erkrankung haben, sollten Sie zum **Arzt** (ein Mediziner, der Menschen behandelt) gehen.
