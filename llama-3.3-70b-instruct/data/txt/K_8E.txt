Hier ist der Text in der Deutschen Leichten Sprache:  **Ummeldung einer Wohnung**  **Wichtige Informationen**  Sie müssen einen Termin vereinbaren, bevor Sie persönlich kommen. Sie können einen Termin online buchen. Es ist auch möglich, einen Termin für mehrere Personen zu buchen.  Bitte buchen Sie nur einen Termin. Wenn Sie mehrere Termine buchen, kann der Fachdienst die anderen Termine löschen. Das ist notwendig, damit andere Menschen auch einen Termin bekommen können.  Wenn Sie einen Termin gebucht haben, aber nicht kommen können, bitte stornieren Sie den Termin rechtzeitig.  Sie können einen Termin hier buchen: https://termin.goettingen.de/  **Was bedeutet Ummeldung?**  Wenn Sie in eine neue Wohnung ziehen, müssen Sie sich ummelden. Eine Wohnung ist jeder Raum, in dem Sie wohnen oder schlafen.  **Wer muss sich ummelden?**  * Wenn Sie jünger als 16 Jahre sind, muss Ihre Familie oder Ihr Betreuer sich ummelden. * Wenn Sie einen Betreuer haben, muss er sich ummelden. * Sie können sich auch von jemandem vertreten lassen, aber Sie müssen das Meldeformular unterschreiben.  **Welche Unterlagen benötigen Sie?**  * Eine Bestätigung von Ihrem Vermieter oder dem Eigentümer der Wohnung. * Ihren Personalausweis oder Reisepass. * Wenn Sie ein Kind sind, benötigen Sie Ihre Geburtsurkunde.  **Kosten**  Die Ummeldung ist kostenlos.  **Fristen**  Sie müssen sich innerhalb von zwei Wochen nach Einzug in die Wohnung ummelden. Wenn Sie das nicht tun, kann es ein Bußgeld geben.  **Kontakt**  Wenn Sie Fragen haben, können Sie die Service-Hotline der Einwohnermeldehalle anrufen. Sie finden alle Informationen hier: https://www.goettingen.de/buergerservice/personen/service-hotline-einwohnermeldehalle-900001583-25480.html
