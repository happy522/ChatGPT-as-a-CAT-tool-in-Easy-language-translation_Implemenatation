Hier ist der Text in der Deutschen Leichten Sprache:  Hilfe für die Seele Manchmal braucht man jemanden zum Reden. Wenn niemand da ist, kann das sehr schlimm sein. Es gibt Hilfe, wenn Sie über Probleme sprechen wollen.  Die evangelische und katholische Kirche bieten eine kostenlose Telefon-Hilfe an. Sie können anonym anrufen, das heißt, niemand erfährt, wer Sie sind.  Die Telefonnummern sind: * 0 800 / 111 0 111 * 0 800 / 111 0 222 * 116 123  Sie können jeden Tag, 24 Stunden lang, anrufen. Es gibt auch Hilfe per E-Mail und im Chat. Sie können sich auf der Webseite der Telefonseelsorge anmelden.  Die Telefonseelsorge ist da, um Ihnen zuzuhören und zu helfen. Sie können über alles sprechen, was Sie belastet.
