**Erste Hilfe: So helfen Sie richtig**

Dieser Text erklärt, wie Sie im Notfall helfen können. Er ist einfach zu verstehen.

**Was ist Erste Hilfe?**

Wenn jemand bei einem Unfall verletzt wird, braucht er schnell Hilfe. Das nennt man "Erste Hilfe". Jeder sollte wissen, wie man Erste Hilfe macht. Wichtige Dinge sind:

- Bringen Sie die verletzte Person an einen sicheren Ort.
- Rufen Sie die Notrufnummer 112 an.
- Legen Sie die Person in die stabile Seitenlage.
- Versuchen Sie, die Person wiederzubeleben.
- Helfen Sie der Person weiter.

**Was tun Sie zuerst nach einem Unfall?**

- Sorgen Sie für Ihre eigene Sicherheit.
- Wenn es ein Auto-Unfall war, stellen Sie ein Warndreieck auf und ziehen Sie eine Warnweste an.
- Bringen Sie die verletzten Personen an einen sicheren Ort. Seien Sie dabei vorsichtig.

**Wie rufen Sie Hilfe?**

- Rufen Sie die Notrufnummer 112 an.
- Sagen Sie dem Notruf, wo der Unfall passiert ist, was passiert ist, wie viele Personen betroffen sind und welche Verletzungen sie haben.
- Warten Sie auf Rückfragen.

**Wie legen Sie eine bewusstlose Person hin?**

Wenn jemand bewusstlos ist, legen Sie ihn in die stabile Seitenlage. Das schützt ihn.

**Wie machen Sie eine Wiederbelebung?**

Wenn jemand nicht mehr atmet, rufen Sie sofort die Notrufnummer 112 an. Dann machen Sie eine Herzdruckmassage und geben Atemspenden. Machen Sie das, bis Hilfe kommt.

**Wie unterstützen Sie eine verletzte Person?**

- Bleiben Sie bei ihr.
- Sagen Sie ihr, dass Hilfe kommt.
- Schützen Sie sie vor neugierigen Menschen.
- Halten Sie ihre Hand.
- Reden Sie mit ihr und hören Sie ihr zu.
