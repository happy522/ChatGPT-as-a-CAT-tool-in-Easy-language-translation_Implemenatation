**Erste Hilfe: So helfen Sie richtig**

Dieser Text zeigt, wie Sie im Notfall helfen können. Er ist leicht zu verstehen.

**Was ist Erste Hilfe?**

Wenn jemand bei einem Unfall verletzt wird, braucht er schnell Hilfe. Das nennt man "Erste Hilfe". Das bedeutet, man hilft sofort. Jeder sollte wissen, wie man Erste Hilfe macht. Wichtige Dinge sind:

- Bringen Sie die verletzte Person an einen sicheren Ort.
- Rufen Sie 112 an.
- Legen Sie die Person in die stabile Seitenlage. Das ist eine Position, die den Körper schützt.
- Versuchen Sie, die Person wiederzubeleben. Das heißt, man versucht, das Atmen und das Herz wieder in Gang zu bringen.
- Helfen Sie der Person weiter.

**Was tun Sie zuerst nach einem Unfall?**

- Sorgen Sie für Ihre eigene Sicherheit.
- Bei einem Auto-Unfall stellen Sie ein Warndreieck auf. Das warnt andere Autofahrer.
- Ziehen Sie eine Warnweste an. Das macht Sie sichtbar.
- Bringen Sie die verletzten Personen an einen sicheren Ort. Seien Sie dabei vorsichtig.

**Wie rufen Sie Hilfe?**

- Rufen Sie 112 an.
- Sagen Sie dem Notruf, wo der Unfall passiert ist.
- Sagen Sie, was passiert ist.
- Sagen Sie, wie viele Personen betroffen sind.
- Sagen Sie, welche Verletzungen sie haben.
- Warten Sie auf Rückfragen.

**Wie legen Sie eine bewusstlose Person hin?**

Wenn jemand bewusstlos ist, legen Sie ihn in die stabile Seitenlage. Das schützt ihn.

**Wie machen Sie eine Wiederbelebung?**

Wenn jemand nicht mehr atmet, rufen Sie sofort 112 an. Dann machen Sie eine Herzdruckmassage. Das heißt, man drückt auf die Brust, damit das Herz wieder schlägt. Geben Sie auch Atemspenden. Das heißt, man gibt dem anderen Luft. Machen Sie das, bis Hilfe kommt.

**Wie unterstützen Sie eine verletzte Person?**

- Bleiben Sie bei ihr.
- Sagen Sie ihr, dass Hilfe kommt.
- Schützen Sie sie vor neugierigen Menschen.
- Halten Sie ihre Hand.
- Reden Sie mit ihr und hören Sie ihr zu.
