Hier ist Ihr Text in Deutscher Leichter Sprache: **Insektenstich** Dieser Text erklärt: Was tun bei einem Insektenstich? Was ist Erste Hilfe? **Was tun bei einem Insektenstich?** Meist sind Insektenstiche in Deutschland nicht gefährlich. Sie können meistens selbst helfen. So gehen Sie vor: - Ist der Stachel von einer Biene noch in der Haut? Nehmen Sie den Stachel vorsichtig heraus. Schieben Sie ihn zum Beispiel mit einer Kreditkarte seitlich aus der Haut. Seien Sie dabei vorsichtig. Ziehen Sie möglichst nicht an der Giftblase. In der Giftblase ist das Gift vom Insekt. Kommt das Gift in die Wunde, kann es mehr wehtun. - Kühlen Sie den Stich. Wickeln Sie ein Kühlpack oder Eiswürfel in ein Tuch. Legen Sie das Tuch auf den Stich. Wichtig: Eiswürfel oder Kühlpacks nicht direkt auf die Haut legen! Die Haut kann sonst Schaden nehmen. - Sie können ein Gel oder eine Salbe gegen Insektenstiche aus der Apotheke benutzen. Das hilft gegen Jucken. - Hitze hilft manchmal auch. Es gibt dafür spezielle Geräte namens Hitze-Stift. Hitze kann den Juckreiz verringern. **Wann müssen Sie den Rettungsdienst rufen?** Manchmal sind Insektenstiche gefährlich. Rufen Sie den Rettungsdienst (112), wenn: - Der Stich ist im Mund oder Hals. - Die Person hat sehr viele Stiche auf einmal. - Die Person reagiert allergisch. **Stich im Mund oder Hals** Das ist sehr gefährlich. Die Stelle kann stark anschwellen. Die Person bekommt vielleicht keine Luft mehr. Rufen Sie sofort den Rettungsdienst (112). Sie können helfen: - Geben Sie der Person ein Eiswürfel zum Lutschen. - Machen Sie kühle Umschläge um den Hals. **Viele Stiche** Die Person wurde von vielen Bienen, Wespen oder Hornissen gestochen? Rufen Sie sofort 112! Viel Gift auf einmal ist gefährlich. **Allergische Reaktion** Reagiert eine Person allergisch? Handeln Sie schnell. Rufen Sie 112! Zeichen für eine Allergie: - Die Person bekommt schlecht Luft - Der Person ist schwindlig - Das Herz schlägt schnell - Die Haut schwillt stark an - Auf der Haut sind rote, dicke Flecken (Quaddeln) Manche Menschen haben ein Notfall-Set für Allergien. Das Set enthält: - Kortison (gegen Entzündung) - Antihistaminikum (gegen Allergie) - Eine Adrenalin-Spritze (hilft dem Kreislauf) So benutzen Sie das Set: - Entfernen Sie den Stachel - Geben Sie Kortison und Antihistaminikum - Entsichern Sie die Adrenalin-Spritze. Setzen Sie sie seitlich auf den Oberschenkel. Auslösen. Wichtig: Atmet die Person nicht mehr? Dann beginnen Sie mit der Wiederbelebung. **Mehr Informationen** Es gibt ein Erklärvideo zu Insektenstichen. Sie finden das Video und mehr Infos im Internet.
