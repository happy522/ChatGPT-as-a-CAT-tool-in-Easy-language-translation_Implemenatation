- Blähungen  

*Reizdarm*  
Beim Reizdarm ist der Darm gereizt.  
Die Verdauung funktioniert nicht gut.  
Pfefferminze entspannt die Darm-Muskeln.  
So werden leichte Krämpfe besser.  

*Kopfschmerzen*  
Pfefferminze hilft bei Kopfschmerzen.  
Das Öl kühlt die Haut.  
So lindert es den Schmerz.  
So machen Sie das:  
- Mischen Sie Pfefferminzöl mit Wasser.  
- Streichen Sie es auf die Schläfen.  
- Massieren Sie sanft.  

Achtung: Pfefferminzöl nicht in die Augen bekommen.  

*Erkältung*  
Pfefferminzöl hilft bei Erkältung.  
Atmen Sie den Dampf ein.  
Das Öl löst den Schleim.  
So können Sie besser Luft holen.  
Pfefferminzöl als Nasenspray hilft bei verstopfter Nase.  
Sie können das Öl auch als Balsam auf die Haut streichen.  

*Muskelschmerzen*  
Pfefferminzöl hilft bei verspannten Muskeln.  
Mischen Sie das Öl mit Wasser.  
Streichen Sie es auf die schmerzende Stelle.  

**Was müssen Sie beachten?**  
Achtung: Pfefferminzöl immer mit Wasser mischen.  
Nicht pur verwenden.  
Sonst können folgende Probleme auftreten:  
- Haut juckt oder brennt  
- Bauchschmerzen  
- Atemnot  

*Bei Reizdarm:*  
Kaufen Sie Pfefferminze als fertiges Medikament in der Apotheke.  
So wirkt es direkt im Darm.  
Fragen Sie den Apotheker um Rat.  

*Bei Kopfschmerzen:*  
Verwenden Sie das verdünnte Öl.  
Streichen Sie es mit den Fingern oder einem Tuch auf die Haut.  

Sie sind unsicher?  
Fragen Sie einen Arzt oder Apotheker.  
Sie zeigen Ihnen, wie Sie Pfefferminze richtig anwenden.  

**Welche Probleme können auftreten?**  
Pfefferminze kann mit anderen Medikamenten zusammenwirken.  
Das kann zu Problemen führen.  
Zum Beispiel:  
- Übelkeit  
- Sodbrennen  
- Brennen unten  
- Trockener Mund  

**Für wen ist Pfefferminze nicht gut?**  
Diese Menschen sollten Pfefferminze nicht nehmen:  
- Babys und kleine Kinder: Menthol kann Atemprobleme machen.  
- Menschen mit Gallen-Problemen: Bei Gallen-Problemen keine Pfefferminze nehmen.  
- Menschen mit Asthma: Pfefferminzöl nicht einatmen. Es kann einen Asthma-Anfall auslösen.
