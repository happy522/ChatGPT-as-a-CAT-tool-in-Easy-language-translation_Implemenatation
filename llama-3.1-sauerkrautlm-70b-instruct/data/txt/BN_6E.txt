Hier ist der Text in der Deutschen Leichten Sprache:  Führungszeugnis _____________________   Wie Sie ein Führungszeugnis beantragen   Das Führungszeugnis ist ein Dokument. Das Bundeszentralregister in Bonn stellt es aus. Sie können das Dokument beantragen.   Es gibt verschiedene Arten von Führungszeugnissen:   - Für private Zwecke (Das Dokument wird Ihnen zugesandt.)  - Für die Vorlage bei einer Behörde (Das Dokument wird an die Behörde geschickt.)  - Erweiterte Führungszeugnisse zur Vorlage bei behördlichen Stellen (kein Privatismus hier).  - Europäische Führungszeugnisse (für EU-Bürger oder Deutsche, die eine weitere EU-Staatsangehörigkeit haben)   Wenn Sie arbeiten wollen, brauchen Sie oft ein Führungszeugnis für private Zwecke. Ihr künftiger Arbeitgeber kann Sie dazu auffordern.   Wenn Sie bei einer Behörde arbeiten wollen oder eine staatliche Erlaubnis beantragen, benötigen Sie ein Führungszeugnis für die Vorlage bei einer Behörde. Das Dokument wird direkt an die Behörde geschickt.   Sie können das Führungszeugnis vorher beim Amtsgericht Braunschweig einsehen, wenn es Eintragungen enthält. Dann geht es aber erst später an die Behörde.   Für das erweiterte Führungszeugnis benötigen Sie ein Anforderungsschreiben von Ihrem Arbeitgeber. Das Schreiben muss folgende Informationen enthalten:   - Name und Adresse des Arbeitgebers  - Eine Bestätigung, dass die Voraussetzungen erfüllt sind  - Ihren Namen, Vornamen, Geburtsdatum und Adresse  - Die gewünschte Belegart  - Die Adresse der Behörde, an die das Dokument geschickt werden soll   Wer kann ein Führungszeugnis beantragen?   Jeder kann ein Führungszeugnis beantragen, der:   - Volljährig ist (ab 18 Jahren) oder von seinen Eltern beantragen lässt (bis 18 Jahren).  - In Braunschweig wohnt (mit Haupt- oder Nebenwohnung).   Wie können Sie ein Führungszeugnis beantragen?   - Online über die Abteilung Bürgerangelegenheiten (ohne Online-Ausweisfunktion)  - Direkt beim Bundesamt für Justiz (mit Online-Ausweisfunktion)  - Persönlich   Welche Unterlagen benötigen Sie?   - Ihren Personalausweis, Reisepass, Kinderreisepass oder Führerschein  - Die Anschrift und das Aktenzeichen einer Behörde (keine Postfachangabe), wenn Sie das Dokument für die Vorlage bei einer Behörde beantragen  - Eine Beglaubigung Ihrer Unterschrift durch einen Notar oder eine Behörde, wenn Sie schriftlich beantragen  - Das Anforderungsschreiben Ihres Arbeitgebers, wenn Sie ein erweitertes Führungszeugnis beantragen   Wie hoch ist die Gebühr?   Die Gebühr beträgt 13,00 EUR.   Wie lange dauert die Bearbeitung?   Ihr Antrag wird sofort bearbeitet und an das Bundesamt für Justiz geschickt.    Was sollten Sie noch wissen?   Bei Online-Beantragung eines gebührenbefreiten Führungszeugnisses ist ein Nachweis über die Gebührenbefreiung erforderlich. Ohne Nachweis wird Ihr Antrag abgelehnt.   wurde letzter Absatt wieder eingefüct   Hinweise und Besonderheiten   Dritte Internetanbieter (z.B.
