Wenn Sie Probleme haben
Sie haben ein Problem? Vielleicht machen Sie sich große Sorgen? Jeder Mensch kann in eine schwierige Situation geraten. Schwierige Situation bedeutet, dass etwas Schlimmes passiert oder dass Sie Probleme haben, die Sie nicht lösen können.

Wer hört mir zu? Wer kann mir helfen?

Die Seelsorge hilft Ihnen
Die Seelsorge ist da, um Ihnen zuzuhören und Sie zu unterstützen. Unterstützen bedeutet, dass jemand Ihnen hilft, Ihr Problem zu lösen. Die Seelsorge ist kostenfrei, was bedeutet, dass Sie keine Gelder zahlen müssen. Sie ist auch anonym, was bedeutet, dass niemand Ihre Personalien kennt.

Wie erreichen Sie die Seelsorge?
Sie können die Seelsorge anrufen unter:
- 0 800 111 0 111
- 0 800 111 0 222
- 116 123

Sie können der Seelsorge auch eine E-Mail schreiben oder den Chat benutzen. Der Chat ist eine Möglichkeiten, mit der Seelsorge über das Internet zu kommunizieren.
