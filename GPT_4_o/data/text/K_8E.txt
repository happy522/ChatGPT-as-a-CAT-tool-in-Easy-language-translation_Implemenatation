Ummeldung einer Wohnung - Leichte Sprache

Wichtige Informationen:
- Sie müssen einen Termin buchen, um persönlich zu sprechen.
- Bitte buchen Sie den Termin online: https://termin.goettingen.de/
- Sie können auch einen Termin für mehrere Personen buchen.
- Bitte buchen Sie nur einen Termin. Mehrere Termine werden gelöscht.
- Wenn Sie einen Termin nicht wahrnehmen können, sagen Sie ihn bitte ab.

Wer muss sich ummelden?
- Wenn Sie in einer neuen Wohnung wohnen, müssen Sie das melden.
- Eine Wohnung ist ein Raum, in dem Sie wohnen oder schlafen.
- Kinder bis 16 Jahre werden von ihrem Vermieter angemeldet.
- Eine Person mit einem Betreuer wird von diesem angemeldet.

Unterlagen:
- Sie brauchen eine Bestätigung vom Vermieter/Eigentümer.
- Einen Personalausweis oder Reisepass. Kinder unter 16 Jahren brauchen eine Geburtsurkunde.

Kosten:
- Die Ummeldung kostet nichts.

Fristen:
- Sie müssen sich innerhalb von zwei Wochen ummelden.
- Wenn Sie die Frist verpassen, gibt es ein Bußgeld.

Kontakt für Fragen:
- Service-Hotline Einwohnermeldehalle: https://www.goettingen.de/buergerservice/personen/service-hotline-einwohnermeldehalle-900001583-25480.html
